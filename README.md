# Beauty E-commerce Product Analytics (SQL Project)

## Project Objective

This project builds a production-style analytics workflow for a beauty e-commerce company using SQL-based data modelling. It analyses acquisition, activation, retention, conversion funnels, revenue performance, and product behaviour.

## Analytics Scope

The project answers key business questions across:

- Acquisition channels performance
- Activation speed after signup
- User retention across cohorts
- Conversion funnel drop-offs
- Revenue trends and AOV
- Product and category performance
- Session engagement behaviour

## Data Modelling Approach

The warehouse follows a layered analytics engineering structure:

raw data → staging → intermediate → marts → dashboard
